# 📊 AutoReport

A Python-based automated sales report generator that reads sales data from multiple sources, performs statistical analysis, detects anomalies, visualizes trends, and generates professional HTML and PDF reports.

---

## 🚀 Features

- Read data from:
  - CSV
  - JSON
  - Excel
  - SQLite
  - REST API

- Statistical Analysis
  - Total Sales
  - Average Sales
  - Highest Sales
  - Lowest Sales
  - Median
  - Standard Deviation

- Trend Analysis

- Anomaly Detection

- Automatic Chart Generation

- HTML Report Generation

- PDF Report Generation

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- SQLite3
- ReportLab
- Requests

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/akashkumaraa193/AutoReport.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```
